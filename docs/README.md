## Recommended Flow

1. Start with the [Starter Playbook](./starter-playbook.md).
2. Use [Template Capabilities](./template-capabilities.md) if more than one template fits.
3. Scaffold with `vp create @brandonxiang:<template-name> my-app`.
4. Use [Starter Recipes](./recipes.md) for the first extension.
5. Use [Troubleshooting](./troubleshooting.md) if setup, test, or build fails.
